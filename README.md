## 🎮 游戏简介

**Pixel Dungeon: Dark Escape** 是一款融合了 **Roguelike、非对称竞技** 和 **AI 生成内容** 的像素风手游。

- 🏰 **AI 生成地牢**：每一局的地图、怪物、物品都由 AI 实时创造，绝不重复
- 👥 **非对称对抗**：人类 vs 恶灵，4-6 人在线对战
- ⚔️ **背叛机制**：组队、背叛、抢夺物资，信任与背叛的心理博弈
- 🎯 **撤离玩法**：找到生命之石，活着逃出地牢
- 💬 **AI NPC 对话**：LLM 驱动的流浪商人、被困幽灵，真实对话交互

## 🎯 背景故事

> "混沌地牢"——一座由远古 AI"造物主"创造的地下迷宫。
>
> 造物主在被封印前，将自己的创造之力散落在地牢的每一块砖石中。从此，这座地牢有了生命——它的地图永远在生长变化，它的怪物不断进化出新的形态，它的宝藏从不会重复出现。
>
> 每隔百年，地牢入口开启。冒险者们蜂拥而入，寻找传说中的 **"生命之石"**——造物主留下的核心碎片，据说能实现持有者的任何愿望。
>
> 但地牢深处，被封印的 **"远古恶灵"** 也在苏醒。它们曾是上一批失败的冒险者，灵魂被地牢吞噬，化为猎杀新人的恶鬼。
>
> 记住，地牢里只有一条规则：**带着生命之石出去的人，才是赢家。**

## 🎯 核心玩法

### 阵营与胜利条件

| 阵营 | 角色 | 人数 | 胜利条件 |
|------|------|------|---------|
| 🧑 **人类** | 冒险者 | 3-5人 | 至少 1 人带着生命之石激活传送石碑并逃出 |
| 👻 **恶灵（鬼）** | 被地牢吞噬的失败者 | 1-2人 | 感染所有人类，或摧毁传送石碑 |

### 单局游戏流程
00:00 所有玩家以人类身份随机出生在地牢各角落
00:00-01:00 无敌保护期，探索周围环境
01:00-03:00 搜刮物资、打怪升级、寻找队友
03:00 地牢中心刷新"幽魂Boss"
03:00-05:00 击败幽魂者可选择化身为"鬼"
05:00-10:00 人类继续搜刮，鬼开始猎杀
10:00 生命之石位置全地图标记
10:00-15:00 争夺生命之石，冲向传送石碑
15:00 地牢开始崩塌（缩圈机制启动）
20:00 地牢完全崩塌，未逃出者全部死亡

### 背叛机制

| 行为 | 后果 |
|------|------|
| 退出队伍 | 可立即攻击原队友 |
| 击杀队友 | 获得"背叛者"标记，全地图可见 |
| 击杀背叛者 | 获得双倍金币奖励 |
| 鬼的感知 | 鬼可精确感知背叛者位置 |

### 物资与装备

| 类型 | 获取方式 | 作用 |
|------|---------|------|
| 💰 金币 | 杀怪、开宝箱、击杀玩家 | 女神像处永久加点 |
| 🧪 药水 | 宝箱、怪物掉落 | 回血、加速、隐身等临时 Buff |
| ⚔️ 武器 | 宝箱、击杀怪物/玩家 | 提升攻击力 |
| 🛡️ 护甲 | 宝箱、击杀怪物/玩家 | 提升防御力 |
| 🔑 钥匙 | 怪物掉落、宝箱 | 开启上锁房间 |
| 💎 生命之石 | 地牢深处 Boss 掉落 | **唯一目标道具，带出地牢即获胜** |

### 鬼的能力

| 能力 | 效果 | 冷却时间 |
|------|------|---------|
| 🔍 灵魂感知 | 显示最近一名人类的方向 | 20秒 |
| 🪤 陷阱 | 在地面放置减速陷阱，持续10秒 | 30秒 |
| 👻 穿墙 | 无视墙壁移动，持续3秒 | 60秒 |
| 😱 恐惧嚎叫 | 附近人类屏幕变暗 + 方向反转3秒 | 90秒 |

每感染一名人类，鬼获得 1 个"灵魂点数"，可升级技能。

## 🤖 AI 深度集成

本项目最大的技术特色是 **AI 深度参与游戏内容生成**，而非简单的预制内容随机拼接。

| 系统 | AI 能力 | 技术方案 |
|------|--------|---------|
| 🗺️ **地图生成** | 每局地牢布局、房间类型、走廊结构完全由算法实时生成 | 波函数坍缩(WFC) + LLM 主题驱动 |
| ⚔️ **物品系统** | 药水效果组合、装备属性、物品描述由 LLM 预生成 | LLM JSON 预生成 + 离线加载 |
| 👾 **怪物生成** | AI 随机组合外观配色 + 行为特性，创造全新变种 | 参数化生成 + 行为树模板 |
| 💬 **NPC 对话** | 流浪商人、被困幽灵由 LLM 驱动，玩家可自由打字对话 | LLM API 实时生成 |
| 🎚️ **难度调节** | AI 监测玩家表现，动态调整怪物数量、物品掉落率 | 规则引擎 + 状态监测 |
